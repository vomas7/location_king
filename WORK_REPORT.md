## 🚀 КАК ЗАПУСТИТЬ СЕЙЧАС

### Если Docker установлен:
```bash
./start_services.sh
```

### Если Docker не установлен:
```bash
./check_and_install_docker.sh
# Затем выйдите и зайдите снова
./start_services.sh
```

### После запуска:
- Frontend: http://localhost
- API Docs: http://localhost:8000/api/docs  
- Keycloak: http://localhost:8080
- Keycloak Admin: http://localhost:8080/admin

## 🔐 ЧТО НУЖНО ДЛЯ PRODUCTION

### Твои действия:
1. **Настроить DNS:** locationking.ru → IP сервера
2. **Получить SSL сертификаты:** Let's Encrypt
3. **Разместить сертификаты:** в папку `ssl/`
4. **Запустить:** `./deploy_production.sh`
