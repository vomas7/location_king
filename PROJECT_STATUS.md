## 🎯 Обзор проекта

**Location King** - геогейссер с космическими снимками. Игрок видит квадрат космического снимка и должен найти его центр на карте OpenStreetMap.

## ✅ ЧТО СДЕЛАНО (90% готовности)

### 🏗️ **Архитектура и инфраструктура**
- [x] Docker Compose конфигурация (nginx, backend, frontend, postgres, redis)
- [x] PostgreSQL с PostGIS для геоданных
- [x] Redis для кэширования
- [x] Nginx как reverse proxy

### 🔧 **Бэкенд (FastAPI)**
- [x] **Модели БД:**
  - `User` - пользователи (интеграция с Keycloak)
  - `GameSession` - игровые сессии
  - `Round` - раунды с геоданными (PostGIS Geometry)
  - `LocationZone` - игровые зоны (PostGIS Polygon)
- [x] **Сервисы:**
  - `satellite_provider.py` - интеграция с Mapbox Satellite API
  - `challenge_generator.py` - генерация случайных точек
  - `geometry_utils.py` - работа с геоданными
- [x] **API эндпоинты (15+):**
  - Сессии: `/api/sessions/*` (старт, получение, завершение)
  - Раунды: `/api/rounds/*` (догадки, подсказки)
  - Зоны: `/api/zones/*` (список, случайная, превью)
- [x] **Аутентификация:** Базовая структура для Keycloak
- [x] **Тестовые данные:** 6+ игровых зон

### 🎨 **Фронтенд**

- [x] **Функциональность:**

  - Лог действий в реальном времени
  - Адаптивный дизайн
- [x] **Интеграция с API:** Полная

## 🚀 КАК ЗАПУСТИТЬ ПРЯМО СЕЙЧАС

### Вариант 1: Docker Compose (рекомендуется)
```bash
cd /home/ilya/location_king
# 1. Получите Mapbox токен на https://account.mapbox.com/
# 2. Добавьте в backend/.env: MAPBOX_ACCESS_TOKEN=ваш_токен
# 3. Запустите: docker-compose up --build
# 4. Откройте: http://localhost
```

### Вариант 2: Ручной запуск (для разработки)
```bash
# Бэкенд
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # добавьте Mapbox токен
python scripts/init_test_data.py
uvicorn app.main:app --reload --port 8000

# Фронтенд (в другом терминале)
cd frontend
python3 -m http.server 8080
# Откройте http://localhost:8080
```
