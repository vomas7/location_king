Проверка исправлений проблем с центром снимка и маркерами

Исправленные проблемы

  • Желтый крестик на центре спутниковой карты ИСПРАВЛЕНО
  • Красный крестик на карте выбора при клике ИСПРАВЛЕНО
  • Центрирование спутниковой карты на реальном центре снимка ИСПРАВЛЕНО
  • Совпадение центра снимка с правильным ответом ИСПРАВЛЕНО

Добавленные изменения:

1. Функция addSatelliteCenterMarker() - добавляет желтый крестик
2. Маркер клика на карте выбора - красный крестик
3. GET /api/mock/rounds/{id} - endpoint для получения центра снимка
4. Автоматическое центрирование на реальном центре снимка
Примечание: После исправлений центр спутниковой карты будет совпадать с правильным ответом. Желтый крестик показывает центр снимка, красный крестик - место клика на карте выбора.

Инструкция по тестированию

  1. Нажмите "Запустить игру"
  2. Нажмите "НАЧАТЬ ИГРУ" в открывшемся окне
  3. Проверьте, что на спутниковой карте есть желтый крестик в центре
  4. Кликните на карте выбора - должен появиться красный крестик
  5. Нажмите "ОТВЕТИТЬ"
  6. Убедитесь, что правильный ответ совпадает с желтым крестиком
